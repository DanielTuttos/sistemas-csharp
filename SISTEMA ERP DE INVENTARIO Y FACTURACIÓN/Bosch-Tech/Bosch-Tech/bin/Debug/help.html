Bosch-tech Project

SIMS is a Standalone system designed for UG& company for sales and inventory management system where a single administrator can use this system for updating the stocks, and also preparing the stock report, sales report and thus to generate bill reports. The front end software that is used here is Visual basic.net which uses easier Graphical user interface and BackEnd as Microsoft SQL Server which is also easy For updating and modifiable database. Since the hardware specification is very basic level the above Front end and Back end software is compatible. The main advantage of this software is to reduce the manual work. The system also does the required calculations, maintains error free reports and validates the final report within a short period of time.
